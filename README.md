# Home Loan Tracker

A personal home loan tracker web app built with React, TypeScript, Chart.js, and a mock JSON backend.

## Features
- Track loan payments, interest, and principal
- Light/dark theme with smooth transitions
- Forecast and what-if analysis
- Responsive, modern UI
- No real backend, no browser storage

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the dev server:
   ```sh
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
